# 背景音乐目录

本目录包含游戏的所有背景音乐文件。

## 文件列表

### 1. stage1.mp3
- **用途**: 第一关背景音乐
- **风格**: 轻快、欢乐
- **时长**: 1-2 分钟（循环播放）
- **情绪**: 积极向上，适合游戏开始
- **参考**: 8-bit 复古游戏音乐，节奏明快

### 2. stage1_boss.mp3
- **用途**: 第一关 BOSS 战音乐

### 3. stage2.mp3
- **用途**: 第二关背景音乐

- **时长**: 1-2 分钟（循环播放）
- **情绪**: 比第一关更紧张，难度提升

### 4. stage2_boss.mp3
- **用途**: 第二关 BOSS 战音乐
- **风格**: 激烈、战斗感强
- **时长**: 1-2 分钟（循环播放）
- **情绪**: 激烈的战斗氛围

### 5. stage3.mp3
- **用途**: 第三关背景音乐

## 音频优化

### 转换为 MP3 (128kbps)

使用 FFmpeg:
```bash
ffmpeg -i input.wav -b:a 128k output.mp3
```

### 标准化音量

使用 FFmpeg:
```bash
ffmpeg -i input.mp3 -af "loudnorm" output.mp3
```

### 确保循环流畅

1. 使用 Audacity 打开音频
2. 检查开头和结尾是否平滑
3. 使用淡入淡出效果
4. 测试循环播放
