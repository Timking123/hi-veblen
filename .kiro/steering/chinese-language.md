# 中文沟通规则

## 语言要求

- **始终使用中文**与用户沟通
- 所有回复、解释、说明都必须使用中文
- 文档、注释、提交信息等也优先使用中文

## 例外情况

以下内容必须保留英文（但需在括号中标注中文）：

- 文件名：如 `GameEngine.ts`（游戏引擎）
- 函数名：如 `handleClick`（处理点击）
- 变量名：如 `isLoading`（加载中）
- 类名：如 `AudioSystem`（音频系统）
- 技术术语：如 TypeScript、Vue、Canvas 等
- 代码中的标识符

## 文档规范

- 文档标题和内容使用中文
- 代码示例中的注释使用中文
- 英文术语首次出现时标注中文翻译

## 示例

```typescript
// ✅ 正确示例
/**
 * 游戏引擎类
 * 负责管理游戏循环和实体
 */
class GameEngine {
  // 启动游戏
  start(): void { }
}

// ✅ 正确的沟通方式
// "我已经创建了 GameEngine（游戏引擎）类..."
// "这个 handleClick（处理点击）函数用于..."
```

## 适用范围

此规则适用于：
- 所有对话回复
- 代码注释
- 文档编写
- 提交信息
- 错误说明
