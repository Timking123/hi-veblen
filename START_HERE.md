## ⚡ 快速开始

### 你需要做的事情：

1. **SSH 连接到服务器**
   ```bash
   ssh root@120.25.234.223
   ```

2. **执行部署命令**
   - 打开 `QUICK_DEPLOY_COMMANDS.md`
   - 复制命令到服务器执行

3. **配置域名 DNS**
   - 登录阿里云域名控制台
   - 添加 A 记录指向 120.25.234.223

4. **等待部署完成**
   - 访问 https://hi-veblen.com

## ⚠️ 重要提示

- 首次部署需要 15-20 分钟
- 确保域名已配置 DNS 解析
- 部署完成后修改管理员密码
